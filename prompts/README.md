# Prompt Templates

This directory contains reusable prompt templates for common development tasks.

## Usage

1. Copy the relevant `.md` file
2. Fill in `[PLACEHOLDER]` sections with your specific requirements
3. Send to your AI coding assistant (OpenCode, Cursor, Copilot, etc.)

## Templates

| Template | When to use |
|----------|-------------|
| `new-component.md` | Creating a new Vue 3 component |
| `new-page.md` | Creating a new page with routing |
| `new-api-service.md` | Creating API service functions |

## Tips

- Always reference `AGENTS.md` for project conventions
- Load the relevant `.opencode/skills/` skill before starting
- Use `bun run build` as the final verification step
